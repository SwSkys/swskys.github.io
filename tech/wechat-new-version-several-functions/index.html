输入法？音乐？微信新版本将会有这几大功能

[1]昨天在微信之夜现场，微信之父张小龙进行了一个半小时的演讲，分享了微信的日活数据、自己对产品的思考、以及对微信未来的一些展望。

我虽然没有看直播，但是在直播结束后，还是仔细的把文字版《微信十年的产品思考》看了两遍，除了微信相关的信息之外，也收获了一些自己的思考。

#ONE 一组数据

到今天，每天——

有10.9亿用户打开微信，3.3亿用户进行了视频通话；

有7.8亿用户进入朋友圈，1.2亿用户发表朋友圈；

有3.6亿用户读公众号文章，4亿用户使用小程序；

还有很多，包括微信支付，企业微信，微信读书，搜索等，就不一一说了。如微信支付，它就像你以前的钱包一样，已经变成了生活常用品。而微信，也真的成为了“一个生活方式”。

如此大的数据，在国内应该可以说是绝无仅有了，其它的国内超级APP的数据都还差了一大截，QQ为7.35亿，抖音为6亿，而支付宝只有4亿。

#TWO 视频号

视频号是微信今年来的工作中心，迭代速度甚至可以和微信的头两年相比，关于视频号张小龙也聊了很多。

视频号是一个人人都可创作的短内容平台。

所以它是公开领域的内容平台，就不能基于微信号来创作了，因为微信账号是私人的ID，所以当你使用视频号的时候，还需要一个公开ID。

而之前微信只有群直播功能而没有真正的直播功能也是因为微信号不足以承担直播的功能。

此外还有视频号的推送机制，视频的播放量，关注，好友推荐，机器推荐的消耗比例，应该是1:2:10。即，一个人应该平均看10个关注的视频，20个朋友赞的视频，100个系统推荐的视频。

#THREE 一些有趣的实验性项目

这些也就是微信未来的展望，张小龙经常会有一些异想天开的想法，之后就会对这些想法进行探索，但是只要思路的方向没有问题，就可以不断改进，最后就会被添加到微信中。

如拍一拍就是这样诞生的，此外在新版本也会有相似好玩的功能的思考点。

  1. 表情，对着对方的微信扔出一个炸弹，对方的屏幕就裂开来，以此满足用户越来越强烈的情绪。
  2. 微信状态，除了名字头像朋友圈视频号，未来微信的名片可能会添加更多东西，让用户通过随便说一句话，随便写几个字来表达自己的状态。
  3. 歌曲，张小龙一直不太满意在微信里面听歌的体验不太好，同时不理解为什么所有播放器都是一个电唱机在转，他希望在听歌的时候可以看到更多的东西，比如把某一首歌变成一个制作得很精美的MV。
  4. 输入法，由于经常收到用户聊天记录信息相关的投诉，而微信从来不会去分析用户的聊天记录，所以张小龙决定推出自己的只能输入法。
  5. 浮窗，目前的浮窗只是提供了一个尚未看完的内容的列表，未来可能更强大。
  6. 直播，当大多数人都有了视频号ID之后，微信就可能会上线真正的直播，附近的直播和人将改被修改为直播和附近。

#END 最后聊聊

对于微信将要推出的输入法和听歌，个人还是比较期待的，毕竟微信推出的产品一向都是小而美的。

另外就是输入法泄露隐私这件事，这个个人是存疑的，因为并不是所有人都使用的是国产输入法，还有大量的使用iOS自带输入法、谷歌输入法或小众输入法的用户，个人对这些输入法的信任是要比微信要高的。
